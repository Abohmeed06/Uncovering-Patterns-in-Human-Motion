# Dataset

The Human Activity Recognition (HAR) dataset is not included in this repository because it exceeds GitHub's upload size limits.

You can download the dataset from the following Google Drive link:

**Dataset:** https://drive.google.com/file/d/1DhSJ2yee4theZLoQWM9SZ1ijrbmGrT_u/view?usp=sharing

After downloading, place the dataset inside the `data/` directory before running the notebook.

Expected folder structure:

```
data/
└── HAR_Master_Dataset.csv
```
